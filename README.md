# CattleModding

Ok so like there is nothing really here

### Controls

| Key | Action |
| ----------- | ----------- |
| ` | Opens Console |
| Space | Jumps when jump is enabled |

### Commands

| Command | Info |
| ----------- | ----------- |
| help | Returns all commands you can run |
| jump | (0/1) sets jumping enabled or disabled |
| god | Toggles god mode |
